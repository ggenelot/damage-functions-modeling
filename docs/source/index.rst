Welcome to my damage functions review project ! 
================================================

Damage functions are used in Integrated Assessment Models (IAMs) to estimate the economic impact of climate change. This project aims to review the damage functions used in the IAMs and to provide a comprehensive overview of the different types of damage functions used in the literature.


.. note::

   This project is under active development.
